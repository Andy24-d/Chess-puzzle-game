# Chess Puzzle Game

Chess Puzzle Game is a learning project built with **Python + Pygame**.

## Description

The repository contains a chess-puzzle style game created for practice with:

- Pygame fundamentals
- Game loop structure
- Board/game-state interaction patterns

This project is intended for educational purposes.

## Initial Status and Objective
Objective: Make a 2D game based in a chess based puzzle in Pygame
Status: Good at algorithms, logic, understandig code. Flaws: Don't know anything about UI
- IA used to focus learning
- Game based on: The Royal Study
